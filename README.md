# Weather Microservice

It is a service developed with .net 5 for weather forecast

# Feature
  - DDD (Domain-driven design)
  - Design Pattern
    - Mediator Pattern
    - Null Object Pattern
    - CQRS Pattern
  - Clean Code
  - .Net 5
  - C# 9
  - Miniprofiler
  - Jwt Authentication
  - Swagger
  - AutoMapper
  - Serilog
  - Sqlite Database
  - Docker
